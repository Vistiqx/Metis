
# Métis Typography System

## Overview

The Métis typography system supports the brand philosophy:

**Ancient Wisdom + Strategic Instrument**

To achieve this, the system uses two complementary font styles:

1. **Authority Serif**
2. **Modern Serif**

---

# 1. Authority Serif

## Purpose

The Authority Serif typeface represents:

- wisdom
- authority
- heritage
- strategic intelligence

It should appear only in high-impact brand moments.

## Usage

Use Authority Serif for:

- MÉTIS wordmark
- Website hero titles
- Major H1 headings
- Intelligence report titles
- Marketing headlines
- Cover pages

## Design Role

Authority Serif establishes **credibility and gravitas**.

It visually connects the brand to:

- classical philosophy
- strategic thought
- intelligence tradition

---

# 2. Modern Serif

## Purpose

The Modern Serif typeface balances readability with sophistication.

It ensures the interface remains:

- readable
- modern
- professional
- analytical

## Usage

Use Modern Serif for:

- H2 / H3 headings
- UI labels
- dashboard titles
- body text
- report content
- documentation
- tables

## Design Role

Modern Serif ensures the platform remains usable during long analysis sessions.

---

# Typography Hierarchy

| Level | Typeface | Use |
|------|------|------|
| Brand / Logo | Authority Serif | MÉTIS wordmark |
| H1 | Authority Serif | Major titles |
| H2 | Modern Serif | Section headings |
| H3 | Modern Serif | Subsections |
| Body | Modern Serif | Content text |
| Data | Modern Serif | Tables and metadata |

---

# Typography Principles

1. Authority Serif should be **used sparingly**.
2. Modern Serif should handle **most readable content**.
3. Maintain strong **contrast and hierarchy**.
4. Avoid mixing more than these two type systems.

---

---

# Summary

The Métis typography system balances:

**Authority + Readability**

Authority Serif communicates the **wisdom and intelligence heritage** of the platform.

Modern Serif ensures **clarity, usability, and analytical precision** across the application and reports.
