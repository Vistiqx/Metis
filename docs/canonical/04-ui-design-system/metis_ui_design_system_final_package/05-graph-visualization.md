# Graph Visualization Standards

## Entity Shapes
- Person: circle
- Organization: hexagon
- Location: diamond
- Event: triangle
- Digital asset: square
- Unknown: outlined circle

## Color Mapping
- selected / focal entity: Magenta Pulse
- active path / relationship emphasis: Solar Orange
- confirmed insight: Signal Yellow
- utilities and focus: Cyber Teal
- anomaly / escalation: Neon Lime
