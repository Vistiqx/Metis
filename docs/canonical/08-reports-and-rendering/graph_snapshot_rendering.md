# Graph Snapshot Rendering

This document defines how graph visuals are embedded into reports.

## Purpose

Graph snapshots translate interactive graph analysis into report-ready visuals.

## Snapshot Rules

Snapshots should:
- preserve primary hierarchy
- preserve Wisdom Gold authority emphasis
- use signal colors only for classification
- avoid label clutter
- include legend context when needed
- remain readable in both digital and print modes

## Rendering Considerations

The rendering system should support:
- cropping rules
- node label density control
- edge visibility balance
- export quality suitable for PDF use
- print-safe variants where needed
