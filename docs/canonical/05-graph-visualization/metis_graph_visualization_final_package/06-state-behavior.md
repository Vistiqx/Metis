# State Behavior

## Node States
- default
- hovered
- selected
- pinned
- flagged
- collapsed cluster
- disabled / hidden

## Edge States
- default
- hovered
- active path
- filtered out
- low-confidence
- temporal playback

## View States
- loading
- empty
- populated
- filtered
- playback
- export-ready

## Rules
Use opacity, stroke, glow, and shape support along with color.
Do not rely on color alone.
