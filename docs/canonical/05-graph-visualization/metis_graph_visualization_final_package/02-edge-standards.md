# Edge Standards

## Relationship Styles
- Confirmed relationship: solid
- Suspected relationship: dashed
- Inferred relationship: dotted
- Temporal relationship: thin pulsing line
- High-confidence active path: brighter weighted line

## Edge Rules
- Avoid thick spaghetti lines across the entire graph
- Weight must indicate importance, not decoration
- Dashed and dotted relationships must remain readable on dark backgrounds
- Directional edges should only appear where the product meaning requires it
