# Layout, Density, and Labeling

## Density Strategy
- Sparse views prioritize discovery
- Medium-density views prioritize relationship tracing
- Dense views require clustering, filtering, or expansion-on-demand

## Label Rules
- Show labels for focal nodes first
- Collapse secondary labels at lower zoom
- Preserve readable spacing between label text and node outline
- Use inspector panels for deep detail rather than overloading the canvas

## Layout Modes
- Force-directed for discovery
- Hierarchical for command chains or flows
- Radial for focal-entity investigations
- Timeline-linked for event sequences
- Geographic overlays only when location meaning is essential
