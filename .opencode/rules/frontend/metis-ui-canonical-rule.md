# Metis UI Canonical Rule

Metis uses a **two-layer UI architecture**.

Layer 1: General Platform Layer\
Layer 2: Signal Intelligence Layer

Signal colors represent intelligence state only and must never recolor
entire UI surfaces.

The interface must maintain **high analyst information density** and
avoid consumer-style dashboards.
