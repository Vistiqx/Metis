
# Métis Website Architecture

Purpose:
The Métis website communicates the platform's intelligence capabilities, builds trust, and converts visitors into demo requests.

Core Sections:
Home
Platform
Solutions
Reports
Visual Analysis
Resources
About
Contact

Hero Section:
Headline describing strategic intelligence from open‑source data.

Platform Page:
Explains architecture, modules, and workflow:
Collect → Correlate → Analyze → Report

Visual Design Rules:
Dark signal‑based backgrounds.
Accent colors only used for signals.
Graph visuals reinforce intelligence narrative.
