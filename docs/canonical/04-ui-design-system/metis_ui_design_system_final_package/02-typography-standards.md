# Métis Typography Standards

Métis uses a two-family hierarchy aligned to the finalized brand system.

## Authority Serif

Authority Serif is reserved for high-importance brand and document moments.

Use for:
- MÉTIS wordmark
- hero titles
- H1 headings
- report titles
- cover pages
- major investigation / briefing titles

Purpose:
- authority
- heritage
- strategic tone
- intelligence briefing character

## Modern Serif

Modern Serif carries the readable analytical layer.

Use for:
- H2 / H3 headings
- UI labels
- body text
- tables
- report content
- metadata
- documentation

Purpose:
- readability
- modern clarity
- long-session usability
- dense information environments

## Hierarchy Rules

- H1 and major cover headings should favor Authority Serif
- readable interface content should favor Modern Serif
- use strong spacing and weight differences to create hierarchy
- avoid introducing additional unrelated type systems
