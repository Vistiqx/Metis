# Entity Shape Language

## Standard Shapes
- Person: circle
- Organization: hexagon
- Location: diamond
- Event: triangle
- Digital Asset: square
- Document / Report: rounded rectangle
- Unknown / Unclassified: outlined circle

## Rules
- Keep shapes simple and legible at small sizes
- Do not rely on icons inside every node at default zoom
- Use shape first, color second, label third
- At far zoom levels, preserve silhouette clarity
