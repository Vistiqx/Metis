# Métis Logo Usage Rules

This document defines how the approved Métis logo system must be used.

## Source of Truth

Use the approved logo visuals already present in:

- `06-brand-assets/Metis-Mockup-Logo.png`
- `06-brand-assets/Metis-Logo-Mockups-01.png`

These mockups define the current approved presentation of:
- primary logo
- icon mark
- dark-background use
- light-background use

No alternate logo should be introduced unless it is approved against those files.

## Approved Logo Roles

### Primary Logo
Use for:
- website header or hero
- report covers
- marketing pages
- presentations
- high-visibility brand moments

### Icon Mark
Use for:
- favicon
- compact app icon
- mobile nav/header
- avatars
- small branded badges

## Clear Space

Minimum clear space around the logo should equal at least the visible height of the capital **M** in the wordmark.

No text, controls, or other graphics may intrude into this area.

## Minimum Sizes

### Primary logo
- Digital minimum width: 140px
- Print minimum width: 1.25in

### Icon mark
- Digital minimum width: 20px
- Print minimum width: 0.25in

## Background Rules

### Dark backgrounds
Preferred treatment:
- approved full-color logo from the official mockup system
- maintain strong contrast
- do not add extra glow or effects beyond the approved visual language

### Light backgrounds
Preferred treatment:
- approved dark/light presentation shown in the logo mockups
- preserve clean contrast
- avoid low-contrast placement on textured light surfaces

## Color Restrictions

The main logo is part of the brand identity layer and must not be recolored using Signal Layer colors.

Do **not** use the main logo in:
- Magenta Pulse
- Solar Orange
- Signal Yellow
- Cyber Teal
- Neon Lime

Signal colors are reserved for findings, graph states, and analytical classification.

## Incorrect Usage

Do not:
- stretch or compress the logo
- rotate the logo
- rebuild the logo manually
- add gradients or shadows not present in the approved system
- substitute signal colors for logo colors
- place the logo on noisy backgrounds without preserving readability

## File Guidance

Until an explicitly approved production vector set exists, the approved existing brand assets should remain the visual source of truth for implementation and documentation.
