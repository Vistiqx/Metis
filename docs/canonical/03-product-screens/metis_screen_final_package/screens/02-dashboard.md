# Dashboard

## Purpose

Summarize active investigations, key intelligence activity, alerts, and recent movement across the workspace.

## Primary Modules

- summary cards
- investigation overview
- alert modules
- quick navigation
- analyst activity context

## Interaction Notes

The screen should support fast analyst orientation, clear hierarchy, and evidence-aware navigation.  
All controls should prioritize clarity over ornament.

## Visual Rules

Use the dashboard as an executive analyst overview, not a marketing page. Gold marks confirmed focal points; signals classify activity.

## Data / Workflow Alignment

This screen must align with:
- the finalized Wisdom Gold authority layer
- the Signal Layer classification system
- investigation-first workflow design
- report-ready analytical outputs where applicable
