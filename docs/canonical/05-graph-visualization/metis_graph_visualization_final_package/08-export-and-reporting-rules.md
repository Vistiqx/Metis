# Export and Reporting Rules

## Export Modes
- live interactive graph
- screenshot export
- report snapshot
- monochrome print-safe view

## Print Rules
- preserve shape semantics
- use black, white, and limited accent if necessary
- labels must remain readable on white paper
- avoid neon effects in print exports

## Report Snapshot Rules
- selected path should remain obvious
- non-relevant nodes should be subdued
- include legend when relationship types are mixed
