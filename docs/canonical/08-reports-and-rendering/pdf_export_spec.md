# PDF Export Specification

This document defines how reports are exported to PDF.

## Supported Modes

- Digital Presentation PDF
- Print-Friendly PDF

## Export Requirements

Exports should support:
- consistent pagination
- readable typography
- embedded graph snapshots
- chart legibility
- appendix inclusion
- metadata for report identity

## Page Behavior

PDF generation should define:
- page size
- margins
- page numbering
- section breaks
- export filename convention

## Asset Rules

Embedded assets should:
- preserve readability
- maintain hierarchy
- avoid low-resolution degradation
- remain useful in both screen and print contexts
