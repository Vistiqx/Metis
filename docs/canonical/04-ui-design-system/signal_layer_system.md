# Signal Layer System

The Signal Layer defines how analytical classification colors are used across the Métis interface and report system.

## Approved Signal Colors

| Signal | Meaning | Hex |
|---|---|---|
| Magenta Pulse | anomaly detection | #FF2D8F |
| Solar Orange | financial activity | #FF7A18 |
| Signal Yellow | relationship strength | #FFD400 |
| Cyber Teal | digital communications | #32D1CC |
| Neon Lime | emerging patterns | #A6FF00 |

## Core Hierarchy

The visual hierarchy is:

1. Environment = dark workspace surfaces
2. Wisdom Golds = confirmed intelligence / primary entities / authority
3. Signal Layer = analytical classification

Signals must never replace the Wisdom Gold intelligence layer.

## Where Signals Appear

Signal colors may be used in:
- graph nodes and edges where appropriate
- alert chips
- findings badges
- timeline flags
- report annotations
- status indicators
- risk and activity labels

## Restrictions

Do not:
- use signals as the primary brand palette
- use signals for the main logo
- allow signals to overpower gold-confirmed intelligence
- assign signal meaning without supporting labels or legend when needed

## Interaction With Reports

### Digital report mode
Signals may be used to classify:
- findings
- risks
- intelligence activity
- communications
- emerging patterns

### Print report mode
Signals must be used with restraint and remain legible on light surfaces.

## Accessibility

Signal meaning must not rely on color alone.
Pair signals with:
- labels
- icons
- legends
- written section headers where needed
