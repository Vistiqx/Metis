# Investigation Report Structure

This document defines the canonical section structure of a Metis investigation report.

## Standard Sections

1. Cover Page
2. Executive Summary
3. Key Findings
4. Methodology
5. Entity Analysis
6. Timeline Analysis
7. Graph Analysis
8. Evidence Appendix
9. Sources Appendix
10. Conclusions / Recommendations

## Report Design Intent

Reports should support:
- executive readability
- analyst traceability
- evidence-backed interpretation
- digital presentation mode
- print-friendly mode

## Section Notes

### Cover Page
Includes report title, investigation context, and branded presentation.

### Executive Summary
Concise overview of findings and strategic implications.

### Key Findings
Highlights the most important conclusions or patterns.

### Methodology
Describes scope, collection approach, and analytical method.

### Entity Analysis
Breaks down relevant entities and their importance.

### Timeline Analysis
Summarizes time-based developments.

### Graph Analysis
Explains key relationships and hidden structures.

### Evidence Appendix
Provides supporting evidence references.

### Sources Appendix
Documents source origin and relevance.

### Conclusions / Recommendations
Provides analyst conclusion, action implications, or next steps.
