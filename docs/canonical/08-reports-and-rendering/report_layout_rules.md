# Report Layout Rules

This document defines visual layout rules for Metis report composition.

## Layout Principles

Reports should feel:
- structured
- premium
- evidence-based
- readable
- professional

## Typography

Use the finalized brand hierarchy:
- Authority Serif for major report titles and top-level hero moments
- Modern Serif for readable body content and section text

## Report Modes

### Digital Presentation Mode
- dark surfaces may be used
- gold authority accents may be used prominently
- signals may classify findings or sections where appropriate

### Print-Friendly Mode
- light pages
- dark readable body text
- restrained gold usage
- minimal visual waste
- strong chart and table readability

## Layout Rules

- maintain strong spacing between sections
- preserve clear hierarchy
- keep charts and graph snapshots readable
- avoid decorative clutter
- use signals only where classification adds value

## Tables and Charts

- must remain readable in export
- must include clear labels
- should preserve hierarchy in both digital and print contexts
