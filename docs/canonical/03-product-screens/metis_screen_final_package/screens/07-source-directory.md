# Source Directory

## Purpose

Provide source inventory, visibility, and source quality management.

## Primary Modules

- source table
- source-type filters
- reliability metadata
- collection timestamps
- source status

## Interaction Notes

The screen should support fast analyst orientation, clear hierarchy, and evidence-aware navigation.  
All controls should prioritize clarity over ornament.

## Visual Rules

Make source trust and provenance easy to assess at a glance.

## Data / Workflow Alignment

This screen must align with:
- the finalized Wisdom Gold authority layer
- the Signal Layer classification system
- investigation-first workflow design
- report-ready analytical outputs where applicable
