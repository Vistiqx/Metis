## Usage Rules

1. Wisdom Golds represent **confirmed intelligence or primary entities**.
2. Signal colors represent **categories of findings and analysis signals**.
3. Gold should remain **rare and authoritative**.
4. Signal colors should never overpower the gold intelligence layer.
5. Dark charcoal / obsidian backgrounds are recommended for digital UI surfaces.
