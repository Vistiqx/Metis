# Screen Inventory

## Core Product Screens

- 01. Login / Access Gate — Secure entry into the platform with strong identity and trust cues.
- 02. Dashboard — High-level operational awareness for investigations, alerts, and source activity.
- 03. Investigations List — Case-style management view for active and archived investigations.
- 04. Investigation Detail — Single case workspace for evidence, notes, linked entities, and outputs.
- 05. Graph Explorer — Primary intelligence correlation environment and emotional center of the product.
- 06. Entity Profile — Structured intelligence object page for people, organizations, locations, and assets.
- 07. Source Directory — Visibility into available sources, quality, freshness, and ingest state.
- 08. Source Detail — Detailed view into source metadata, freshness, related entities, and usage.
- 09. Timeline View — Temporal analysis surface for events, correlation, and sequence review.
- 10. Reports List — Management surface for all generated and draft reports.
- 11. Report Builder — Analyst-grade composition surface for polished intelligence output.
- 12. Alerts Queue — Triage and escalation surface for new and ongoing alert events.
- 13. Analytics — Operational analytics for throughput, source utility, and reporting trends.
- 14. Settings / Administration — Workspace, users, connectors, theme, audit, and governance controls.
