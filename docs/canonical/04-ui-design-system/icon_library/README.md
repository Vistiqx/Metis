# Icon Library

This directory defines the approved icon set for Métis.

## Purpose

The icon system supports:
- navigation
- metadata
- findings chips
- card headers
- legends
- buttons when paired with text
- analytical labeling

## Style Rules

Icons should be:
- serious
- geometric
- restrained
- readable at small sizes
- consistent with a premium intelligence platform

## Visual Standard

Recommended implementation standard:
- SVG
- 24px grid
- 2px stroke
- clean corners
- no playful decoration

## Required Icon Subjects

- investigation
- entity
- organization
- person
- location
- document
- evidence
- timeline
- alert
- report
- settings
- analytics
- graph
- source

## Important Rule

This directory documents the icon system only.
Do not invent icons that conflict with the core brand tone or Signal Layer meanings.
