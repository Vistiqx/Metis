# Metis Canonical UI Specification

This document defines the **visual design language** for the Metis intelligence analysis interface.

## Visual Design Principles

Metis must resemble:

• a secure intelligence workstation  
• a strategic analysis environment  
• a mission operations console  

It must **not resemble**:

• consumer SaaS dashboards  
• marketing dashboards  
• generic admin panels  

---

# Core Aesthetic

Metis uses a **dark intelligence aesthetic**.

Primary characteristics:

• deep intelligence background  
• luminous wisdom gold accents  
• subtle glass-like analytical panels  
• restrained contrast for analyst endurance  
• high readability during long investigative sessions  

---

# Surface Styling

Panels should use:

• dark layered surfaces  
• subtle edge illumination  
• soft shadow separation  

Panels must avoid:

• bright flat surfaces  
• heavy gradients  
• decorative UI effects  

---

# Wisdom Gold Accent System

Wisdom Gold accents define **platform authority surfaces**.

Used for:

• navigation highlights  
• key headings  
• panel edges  
• icons representing intelligence nodes  

Wisdom Gold must **not dominate entire surfaces**.

---

# Typography

Typography must maintain **analyst readability**.

Headings:

• refined analytical style  
• medium weight  

Body text:

• neutral sans-serif  
• compact line height  

---

# UI Density

Metis is an **analyst-first platform**.

UI must emphasize:

• compact layouts  
• dense information displays  
• minimal wasted whitespace  

Avoid:

• oversized cards  
• marketing-style layouts  
• decorative whitespace  

---

# Primary UI Regions

Every major view must contain:

1. Navigation layer  
2. Analytical workspace  
3. Supporting intelligence panels  

These regions must remain visually consistent across pages.
