# Motion Language

Motion in Métis should support analytical clarity, not spectacle.

## Motion Characteristics

Motion should feel:
- restrained
- precise
- calm
- purposeful
- informative

## Approved Motion Areas

### Graph expansion
Use controlled reveal when showing connected entities.

### Panel transitions
Use quick, smooth transitions that preserve workflow continuity.

### Signal activity
Signals may pulse lightly where they represent active or recent status.

### Loading
Loading indicators should be minimal and professional.

## Motion Restrictions

Avoid:
- bounce effects
- playful animation
- exaggerated spring motion
- flashy glow animation
- slow cinematic transitions that interrupt analysis

## Timing Guidance

- hover states: very short
- panel transitions: short and smooth
- graph reveal: moderate and controlled
- loading states: minimal repetition

## Accessibility

Motion must respect reduced-motion preferences.
No essential state should depend on animation alone.
