# Chart System

This document defines chart standards for Métis outside of the node-link graph system.

## Supported Chart Types

- bar charts
- line charts
- area charts
- stacked bars
- summary metric cards
- frequency charts
- confidence/risk distributions
- transaction or activity trend views

## Chart Philosophy

Charts should feel:
- analyst-friendly
- dense but readable
- restrained
- structured
- appropriate for executive reporting

Avoid decorative dashboard excess.

## Color Rules

### Wisdom Gold usage
Use Wisdom Golds for:
- primary or confirmed metrics
- featured values
- executive emphasis
- core totals

### Signal Layer usage
Use Signal Layer colors only when the chart is expressing analytical classification such as:
- anomalies
- financial activity
- relationship strength
- communications
- emerging patterns

## Report Compatibility

All chart styles must work in:
1. digital report mode
2. print-friendly report mode

Charts included in reports should:
- remain legible in print
- preserve hierarchy
- use clear labels
- avoid unnecessary visual clutter

## Titles and Labels

All charts must include:
- clear title
- explicit metric or unit
- timeframe when applicable
- source or evidence context when needed
