# Interaction and Motion

## Motion Use
- selected node pulse
- relationship expansion reveal
- hover focus bloom
- cluster open/close transition
- path trace animation

## Avoid
- constant ambient flashing
- theatrical movement on every interaction
- infinite pulsing across whole graphs

## Interaction Rules
- click selects
- double click expands neighborhood
- hover previews relationship context
- keyboard navigation must support stepping through graph elements
- inspector remains synchronized with graph focus
