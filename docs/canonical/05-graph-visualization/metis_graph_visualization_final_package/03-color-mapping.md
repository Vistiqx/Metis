# Color Mapping

## Primary Signal Mapping
- Selected / focal entity: Magenta Pulse
- Active path / investigation trail: Solar Orange
- Confirmed insight / validated cluster: Signal Yellow
- Utility / hover / focus / infrastructure: Cyber Teal
- Anomaly / escalation / watch condition: Neon Lime
- High-risk destructive warning: Warning Red

## Neutral Colors
- Graph canvas: Midnight
- Inspector panels: dark surface
- Secondary labels: muted neutral
- Non-focused graph elements: reduced-opacity neutral

## Color Rules
- Keep one dominant emphasis color per user action
- Never use all signal colors at equal intensity simultaneously
- Neon Lime should remain rare and meaningful
