# Color Standards

## Core Brand Colors
- Midnight
- Magenta Pulse
- Solar Orange
- Signal Yellow
- Cyber Teal
- Neon Lime

## Rules
- Do not flood panels with accent colors.
- Use signal colors as meaning, not decoration.
- Keep one dominant accent per view.
- Reserve Neon Lime for anomalies and alerts.
- On white backgrounds, use black or deep ink structure with signal nodes or small signal accents.

## Logo on White
Preferred variants:
1. Black linework + signal nodes
2. Black monochrome
3. Black wordmark + Magenta Pulse accent
