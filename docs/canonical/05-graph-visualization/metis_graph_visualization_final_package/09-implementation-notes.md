# Implementation Notes

## Recommended Build Features
- node shape primitives
- edge style primitives
- zoom-aware labels
- clustering support
- path highlighting
- temporal playback support
- export-safe rendering mode

## Engineering Priorities
1. performance
2. readability
3. interaction clarity
4. export fidelity
5. motion polish

## Non-Negotiables
- graph must remain legible
- shape language must persist across zoom states
- selected paths must be visually obvious
